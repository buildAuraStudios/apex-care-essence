# 🏥 Apex Clinic Website

A modern, responsive, and patient-focused clinic website designed to establish trust and deliver a seamless digital experience. Built with clean design, intuitive navigation, and optimized performance, this template is ideal for healthcare professionals and medical practices looking to strengthen their online presence.

---

## ✨ Overview

Apex Clinic combines professional aesthetics with user-friendly functionality to create a welcoming online experience for patients. From showcasing medical services to simplifying appointment bookings, every section is thoughtfully designed to inspire confidence and improve accessibility.

---

## 🚀 Features

- Modern and professional UI/UX
- Fully responsive across all devices
- Clean and accessible interface
- Interactive hero section
- Medical services showcase
- Doctor profiles
- Online appointment section
- Patient testimonials
- FAQ section
- Contact information & location
- Optimized for speed and performance

---

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui
